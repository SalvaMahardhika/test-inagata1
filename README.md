# E-commerce Inventory API

RESTful API untuk sistem manajemen inventaris toko e-commerce.

Project ini dibuat menggunakan:
- Laravel 11
- MySQL
- RESTful Architecture

---

## 📌 Fitur Utama

- CRUD Produk
- CRUD Kategori
- Pencarian Produk (by name & category)
- Update Stok Produk
- Perhitungan Total Nilai Inventory
